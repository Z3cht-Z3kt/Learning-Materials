# VS Code Shortcuts & Terminal Commands Cheat Sheet

## 🔥 CORE VS CODE SHORTCUTS

### Navigation
- Ctrl + P → Quick open file
- Ctrl + G → Go to line
- Ctrl + Tab → Switch tabs
- Ctrl + Shift + P → Command Palette

### Editing
- Shift + Alt + ↓ / ↑ → Copy line
- Alt + ↓ / ↑ → Move line
- Ctrl + Shift + K → Delete line
- Ctrl + Z / Ctrl + Y → Undo / Redo

### Multi-cursor
- Alt + Click → Add cursor
- Ctrl + D → Select next match
- Ctrl + Shift + L → Select all matches

### Code Navigation
- F12 → Go to definition
- Alt + F12 → Peek definition
- Alt + ← → Go back

### Formatting & Refactoring
- Shift + Alt + F → Format code
- F2 → Rename symbol
- Ctrl + . → Quick fix

### Terminal
- Ctrl + ` → Open terminal
- Ctrl + Shift + ` → New terminal

---

## ▶️ RUN & DEBUG

- Ctrl + F5 → Run without debugging
- F5 → Start debugging
- Shift + F5 → Stop debugging
- Ctrl + Shift + F5 → Restart debugging
- F9 → Toggle breakpoint
- F10 → Step over
- F11 → Step into
- Shift + F11 → Step out

---

## ⚡ TERMINAL COMMANDS (COMMON)

### Navigation
- cd folder_name → Enter folder
- cd .. → Go up one level
- dir (Windows) / ls (Linux/Mac) → List files

### File Management
- mkdir folder_name → Create folder
- del file.txt (Windows) / rm file.txt → Delete file
- copy file1 file2 (Windows) / cp file1 file2 → Copy file
- move file1 file2 (Windows) / mv file1 file2 → Move/rename file

### Git Basics
- git init → Initialize repo
- git add . → Stage all changes
- git commit -m "message" → Commit changes
- git status → Check status
- git log → View history

### Python
- python file.py → Run script

### Java
- javac File.java → Compile
- java File → Run

---

## 🧠 WORKFLOW TIPS

- Use keyboard over mouse
- Use Ctrl + P instead of clicking files
- Use search instead of scrolling (Ctrl + F / Ctrl + Shift + F)
- Keep terminal open
- Learn shortcuts gradually (5 at a time)
